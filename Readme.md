# Personal Portfolio

A personal portfolio website built with HTML, CSS, JavaScript and ionicons.

## Features

* Responsive design
* Custom scrollbars
* Custom select with filter functionality
* Custom modal with testimonials
* Custom navigation menu
* Custom contact form with validation
* Custom page navigation with active class
* Custom favicon

## Technologies Used

* HTML
* CSS
* JavaScript
* ionicons

## Installation

1. Clone the repository
2. Open the index.html file in a web browser
